CS291 - Computer Science II

Description

Data structuring using structured variables, stacks, queues, linked lists, and trees; recursive procedures; applications to sorting and merging. The class emphasizes theoretical understanding of data structures, their implementation, and the object-oriented viewpoint. Assignments will contain significant programming projects along with programming-free questions that explore how data structures work.

Prerequisites

CS 172, MATH 141 OR MATH 225

Learning Objectives

At the end of the course, the student should:

  1. Be able to analyze asymptotic running time of a given algorithm.
  2. Be able to implement and use linear data structures like linked lists, stacks and queues.
  3. Be able to implement and use search structures and algorithms like binary search.
  4. Be able to implement and analyze different sorting algorithms.
